# Bartender de Bolso

Este projeto é para a construção de uma aplicação que sugere e informa receitas de cocktails de seu banco de dados, com base nos ingredientes informados.



# Pré-requisitos
**Node.js**

# Tecnologias
- Express.js
- Nodemon

# Preparar
- Git clone
- Npm install

# Executar
- Npm run dev
